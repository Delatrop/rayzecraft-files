📁 IMAGENS NECESSÁRIAS PARA O LAUNCHER

Coloque suas imagens personalizadas nesta pasta com os seguintes nomes:

1. 🖼️ icon.ico
   - Tamanho: 32x32 ou 64x64 pixels
   - Formato: .ico
   - Descrição: Ícone do launcher (aparece na barra de tarefas)

2. 🖼️ logo.png
   - Tamanho: 200x80 pixels (recomendado)
   - Formato: .png (fundo transparente)
   - Descrição: Logo principal do seu servidor

3. 🖼️ background.jpg
   - Tamanho: 800x600 pixels ou maior
   - Formato: .jpg ou .png
   - Descrição: Imagem de fundo da janela principal

4. 🖼️ splash.png
   - Tamanho: 300x300 pixels
   - Formato: .png (fundo transparente)
   - Descrição: Imagem do splash screen (tela de loading)

🔧 COMO USAR:
1. Substitua esses arquivos pelas suas imagens
2. Mantenha os nomes exatos (icon.ico, logo.png, etc.)
3. No Visual Studio, clique com botão direito na imagem
4. Selecione "Properties"
5. Altere "Build Action" para "Resource"
6. Compile o projeto (F6)

⚠️ IMPORTANTE:
- Mantenha os tamanhos recomendados para melhor qualidade
- Use PNG para imagens com transparência
- Use JPG para imagens sem transparência (menor tamanho)
